    • The easiest way to install is to use OPAM. Install Opam, then, as a regular user, fetch and install an OCaml version and install Stog. The whole can be done with the following opam commands:

    $ opam init --comp=4.02.1 # or 4.02.2 $ opam update $ opam install stog

    To build Stog, you will need:

    • OCaml >= 4.03.0,
    • Ocf >= 0.5.0,
    • Ptime >= 0.8.2,
    • Uutf >= 1.0.0,
    • ocaml-uri >= 1.9.2,
    • ppx_blob >= 0.1,
    • Xtmpl >= 0.16.0,
    • Lwt >= 2.5,
    • Higlo >= 0.6.

    These additional libraries are required to compile and install the preview server:

    • Js_of_ocaml >= 2.6,
    • Xmldiff >= 0.5.0.
    • OCaml-Websocket >= 2.6,
    • Cryptokit >= 1.11,
    • Ojs-base >= 0.5.0,
    • Ppx-deriving-yojson >= 3.0.

    To force the compilation of the stog server, you can use this commande line: $ opam install lwt js_of_ocaml sha xmldiff websocket ojs-base ppx_deriving_yojson stog

    You will need to have the following tools installed to use some functions:

    • highlight to use syntax highlighting for languages not supported by Higlo,
    • LaTeX and dvisvgm to embed LaTeX in your pages.

    Get the sources from git repository:

    $ git clone https://github.com/zoggy/stog.git

    Or pick one of the releases:

    Just type

    ./configure && make && make install

    This will install stog, stog.byte, stog-ocaml-session, mk-stog, mk-stog.byte and mk-stog-ocaml-session in the same directory as ocamlc. This will also install the stog library and the stog.asy, stog.disqus, stog.extern, stog.dot, stog.markdown, stog.multi-doc, stog.nocaml, stog.rel-href, stog.sitemap plugins with findlib, and (eventually) stog.server and stog.multi-server.
